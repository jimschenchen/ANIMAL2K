# ANIMAL2K
 ANIMAL2K is an dataset colleting animal images with their downsamples (only 2X) from DIV2K and Flickr2K datasets. It contains 327 animals images in total.

## DIV2K
https://cv.snu.ac.kr/research/EDSR/DIV2K.tar

## Flickr2K
